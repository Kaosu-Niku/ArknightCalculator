Custom 適配案例總覽
=====================

更新日期：2026-06-20

用途
----
快速確認目前哪些幹員、技能、天賦與模組分支已進入 custom 處理。
本文件中的幹員與技能名稱保留 zh_CN JSON 原名，以便直接對照程式規則。

標記說明
--------
- 一般列出：已有 custom 效果重映射、傷害流、條件、段數或時間規則。
- [僅排除]：沒有額外增加傷害，只阻止語意錯誤的原始 key 進入通用公式。
- 分支規則：對該分支所有符合條件的模組自動生效，不逐一綁定幹員名稱。

統計
----
- 技能 custom：199 個技能案例，149 位幹員。
- 天賦 custom：117 位幹員（83 位有明確效果規則；部分幹員同時也有錯誤 key 排除）。
- 模組 custom：依分支統一處理攻擊倍率、最終增傷、穿透、攻速與額外傷害流。

天賦 custom 目前處理的主要類型：
- 攻擊力／攻速疊層與滿層值。
- 機率攻擊倍率的期望值。
- 條件脆弱／易傷回饋。

- 排除只影響其他友軍、替身、召喚物、反傷或限時部署增益的 key。


三、模組與分支 Custom
---------------------

基礎分支特性
------------
- 解放者：攻擊力蓄力加成。
- 獵手、散射手：分支攻擊倍率。
- 御械術師：浮游單元額外法術傷害流。
- 投擲手：第二段物理傷害流。

裝備即生效的模組特性
----------------------
- 解放者：攻擊力加成。
- 獵手、散射手：攻擊倍率。
- 劍豪：最終傷害與固定無視防禦。
- 炮手：固定無視防禦。
- 中堅術師：固定法抗穿透。
- 要塞：攻擊速度。
- 御械術師、投擲手、領主：第二獨立傷害流。
- 馭法鐵衛：技能期間額外法術傷害流。
- 投擲手：模組追加第三段攻擊。

受條件／機率開關控制的模組特性
------------------------------
- 攻擊力：尖兵、吟遊者、處決者、行商。
- 攻擊倍率：衝鋒手、戰術家、無畏者、強攻手、教官、速射手、攻城手、炮手、
  塑靈術師、撼地者、陷阱師、回環射手。
- 最終傷害：術戰者、神射手、攻城手、轟擊術師、陣法術師、本源術師。
- 攻擊速度：無畏者、術戰者、鬥士、領主、速射手、秘術師。

機率模組會讀取 prob、attack@prob、talent@prob 或 buff_prob，開啟條件效果後按期望值計算。


四、共用公式能力
---------------

目前 custom 可輸出的特殊欄位包含：
- CHANGE_attackType、CHANGE_OTHER_attackType
- atk、FLAT_atk、atk_scale、damage_scale
- def、def_penetrate_fixed、magic_resistance
- base_attack_time、attack_speed
- ATTACK_COUNT、times、MAIN_times、attack@trigger_time
- CHANGE_duration、OTHER_atk_scale、OTHER_base_attack_time、OTHER_times、OTHER_duration
- DISABLE_TRAIT_EXTRA（停止普通攻擊時一併停用依附普通攻擊的模組額外傷害流）

實際規則來源
------------
- 技能：src/model/skillEffectRules.js、src/model/SkillCustomCalculator.js
- 天賦：src/model/talentEffectRules.js
- 模組：src/model/uniequipTraitRules.js
- 傷害與排程：src/model/SkillCalculator.js、src/model/SkillTotalFormula.js
- 全量稽核：scripts/audit-custom-coverage.mjs（npm run custom:audit）
