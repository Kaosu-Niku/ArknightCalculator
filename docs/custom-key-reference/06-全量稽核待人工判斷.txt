全量 JSON 稽核：待人工判斷案例
===============================

更新日期：2026-06-20

說明
----
本清單只保留無法由 JSON 文案安全決定單一目標傷害的案例；已能確定的停攻、
多段、巢狀倍率、機率期望值與錯誤通用 key 已直接加入 custom。

技能時序與目標假設
------------------
- 小满「乡音沉沉」：前 5 秒停攻，之後才以加速攻擊；目前公式需要主傷害流的
  延遲起始時間，才能同時正確計算總傷與整段技能 DPS。
- 澄闪「澄净闪耀」：已確認每個浮游單元各自獨立累積增傷層數；目前公式只有
  共用傷害流，待加入浮游單元各自暖機與自爆期望值後再處理。
- 缄默德克萨斯「阵雨连绵」：部署瞬間的一次傷害與後續二連擊段數不同；現行
  ATTACK_COUNT 會作用於所有傷害流，需先增加每條傷害流獨立段數。
- 琳琅诗怀雅「千金一掷」：持續輸出可算二連擊，但主動關閉時的金幣攻擊取決於
  人工關閉時間與實際金幣數；目前只計持續輸出。
- 丰川祥子「满月的舞会」：Fever 的二連擊需確認要按全技能覆蓋、指定覆蓋率，
  或只在條件開關開啟時視為全程生效；目前所有 Fever 內容均保留。

依敵方或外部行為觸發
--------------------
- 菲莱「冥河诅咒」、年「铜印」、棘刺「护身尖刺」屬受擊反擊，目前總傷為 0。
- 召喚物、替身、自動裝置獨立行動造成的傷害，仍依既定政策不併入幹員技能總傷。

已決策並完成
------------
- 罗小黑「掠光尾影」：拆成「切換前／切換後」兩列。
- 火龙S黑角「居合拔刀气刃斩」：視為實戰必定觸發一次，計算四段 150% 物理傷害。
- 雷狼龙S空爆「高压回填斩」：按五連擊與十連擊完整交替週期的平均 7.5 段，
  每段同時計算 35% 法術附傷。
- 炎狱炎熔「狱火之环」：條件關閉計單環，開啟計雙環重疊。
- 蕾缪安「礼炮·强制追思」：每發採爆心最高倍率 450%。
- 娜仁图亚「旋刃」、林「玲珑」、荒芜拉普兰德「慵怠者悲鸣」、山「横扫架势」、
  银灰「雪境生存法则」、赫德雷「余烬重荷」：各拆成「切換前／切換後」兩列。

尚需公式能力
------------
- 天賦固定 DOT：蓝毒、深巡等，需要天賦可建立獨立傷害流。
- 以敵方防禦為基準的附傷：刻俄柏「剥壳」，不能套用一般自身攻击力倍率。
- 天賦固定 DOT：棘刺、隐德来希等，同樣等待天賦獨立傷害流。
- 每條傷害流獨立段數與延遲起始時間。
- 元素損傷／元素傷害目前不與物理、法術、真實傷害混算。

稽核指令
--------
npm run custom:audit -- --summary

完整輸出會列出尚未覆蓋案例、命中的風險旗標、黑板 key 與技能／天賦文案。
掃描器採高召回設計，候選數量不代表錯誤數量；隊伍增益與純防禦效果也可能被列出。
