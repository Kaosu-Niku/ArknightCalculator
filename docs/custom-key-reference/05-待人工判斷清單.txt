Custom 適配待人工判斷清單
=========================

更新日期：2026-06-19

說明
----
以下項目不是無法辨識 JSON，而是存在兩種以上合理的計算口徑。
請先決定各組的統一原則；確認後可批次套用到同類幹員，不必逐位回答。


一、機率觸發的攻擊倍率／額外傷害
--------------------------------

狀態：已確認。
統一口徑：關閉時不觸發；開啟時按機率計算期望值。
加法型加成使用 value * prob，完整倍率使用 1 + (value - 1) * prob。

代表幹員：
- 紅豆「蠻力穿刺」：技能中 30% 機率，當次攻擊力 +110%。
- 隕星「爆破附著改裝」：30% 機率，當次攻擊力 +60%。
- 月見夜「要害瞄準」：20% 機率，攻擊力提升至 160%。
- 黑「破甲箭頭」：20% 機率，攻擊力提升至 160% 並降防。
- 風笛「精密填彈」：28% 機率，攻擊力提升至 130%。
- 艾拉「正中靶心」：30% 機率造成 160%，對雷鳴地雷目標必定觸發。
- 山「巨力重拳」：20% 機率提升至 165%，並降低敵人攻擊力。
- 鐧「天生的武者」：10% 機率提升至 165%。
- 阿「混合藥物射擊」：隨機四種效果之一，其中一項提升至 150%。

首批已接入：紅豆、隕星、月見夜、Stormeye、風笛、山、鐧、黑、戰車、鴻雪，
以及帶機率 blackboard 的條件模組；後續同類 custom 沿用相同 helper。

統一口徑：關閉時不觸發；開啟時，條件成立就必定觸發的效果完整計入。

代表幹員：
- 鐸鈴：生命高於 50% 時 +22% 攻擊力。
- 布洛卡：阻擋兩名以上敵人時增加攻防。
- 折光：範圍內存在凋亡爆發敵人時 +21% 攻擊力。
- 水月：範圍內存在半血以下敵人時 +12% 攻擊力。
- 苦艾：攻擊生命低於 40% 的敵人時傷害 +30%。
- 戴菲恩：攻擊生命高於 80% 的敵人時傷害 +20%。
- 燧石：攻擊未被自身阻擋的敵人時傷害 +45%。
- 森蚺：生命高於一半時傷害提升至 117%。
- 歌蕾蒂婭：攻擊重量不高於 3 的敵人時提升至 136%。
- 瑕光：攻擊沉睡目標時提升至 144%。

技能、天賦、模組共用同一個「條件與機率效果」開關；計算紀錄會標示目前狀態。


三、對敵人施加脆弱／易傷是否回饋自身技能
------------------------------------------

狀態：已確認。
統一口徑：採 C。只有「條件與機率效果」開啟時，才把幹員施加的脆弱／易傷回饋到自身技能傷害；機率型效果按期望值計算。

代表幹員：
- 初雪、巫戀：敵人低生命時獲得脆弱。
- 鈴蘭：停頓敵人獲得脆弱。
- 焰影葦草：30% 機率施加法術脆弱。
- 濯塵芙蓉：攻擊後施加法術脆弱。
- 慑砂、天火：特定敵人受到的物理／法術傷害提高。
- THRM-EX、正義騎士號、PhonoR-0：部署後提供限時脆弱。

已接入：初雪、巫戀、鈴蘭、焰影葦草、濯塵芙蓉、慑砂、天火、THRM-EX、正義騎士號與 PhonoR-0。

五、額外獨立傷害流
------------------

狀態：已確認統一原則，個別幹員仍以 custom 判斷。
統一口徑：只計算由本體技能攻擊直接觸發，且能從 JSON 得到明確倍率與次數／間隔的額外傷害。

代表幹員：
- 刻俄柏：額外造成敵方防禦力一定比例的法術傷害。
- 忍冬：首次攻擊後，對同目標額外造成 30% 攻擊力法術傷害。
- 仇白：對停頓／束縛敵人額外造成 43% 法術傷害。
- Logos：40% 機率對另一目標造成 65% 額外法術傷害。
- 歸溟幽靈鯊：替身每秒造成 40% 攻擊力法術傷害。
- 黑鍵：儲存攻擊能量與單目標額外傷害。
- 瑪恩納：卡西米爾幹員受擊時反彈真實傷害。
- 新約能天使：友方消耗彈藥時機率召喚轟炸。

明確排除：歸溟幽靈鯊替身傷害、瑪恩納受擊反傷，以及其他依敵方攻擊、替身或召喚物自行行動的傷害。


六、群體／職業增益是否視為會加到幹員自己
--------------------------------------

狀態：已確認。
統一口徑：只要文案與遊戲效果允許幹員本人獲得增益，就計入本人；明確只作用其他單位或本人不符合條件者排除。

代表幹員：
- 艾雅法拉：所有術師攻擊力提升，包含自身。
- 推進之王：所有先鋒攻防提升，包含自身。
- 號角：所有重裝攻擊力提升，包含自身。
- 黑：場上另有狙擊時，所有狙擊攻擊力提升，包含自身。
- 杜賓：所有三星幹員攻擊力提升，不包含四星的自己。
- 詩懷雅：周圍近戰友方提升，是否包含自身取決於遊戲效果定義。
- 城堡-3、嘉維爾：部署後限時提升一組友方單位。

特例：嘉維爾的醫療增益只有部署後短時間生效，因此不計；杜賓不屬於三星幹員，詩懷雅不在自身周圍八格，兩者也不計自身。


七、逐漸提升或多次施放後強化
----------------------------

狀態：已確認。
統一口徑：一律取技能可達到的最高／最終穩定階段，不另外建立首次施放列。

代表技能／天賦：
- 棘刺三技能：第二次施放後永久強化。
- 薇薇安娜三技能：第二次及以後變成三連擊並延長時間。
- 暖機、逐漸提升攻擊力、技能末段爆發等技能。
- 玫拉：首次釋放後，後續技能傷害提高。

已接入棘刺、薇薇安娜、艾雅法拉、號角、怒潮凜冬、遙等多次施放技能，以及能從 JSON 明確取得上限的逐漸增傷／攻速技能。


八、治療技能是否需要計算治療總量
--------------------------------

目前護佑者等治療技能在傷害表中總傷為 0，這符合「不造成傷害」。
但大量技能的 atk、heal_scale 實際用於治療。

需要決定：
A. 傷害表維持 0，不新增治療計算。
B. 同一表格增加「技能總治療／HPS」。
C. 建立獨立治療表。

我的建議：C，避免把傷害與治療的目標數、溢出治療、治療間隔混在同一公式。
